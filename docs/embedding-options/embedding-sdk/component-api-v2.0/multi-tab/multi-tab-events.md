# Events

The events supported for programmatic multitab dashboards are identical to those used for loading standard dashboards. For a complete list of supported events, please refer to the [dashboard events](/embedding-options/embedding-sdk/component-api-v2.0/dashboard/dashboard-events/) documentation.
