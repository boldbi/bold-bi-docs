# Dashboard 
In Dashboard mode, the Unified AI Agent delivers brief summaries of dashboards, emphasizing key trends and insights. select **Dashboard** from the dropdown menu to access this feature.

## Dashboard Summary
Unified AI Agent can analyze a dashboard and provide a summarized overview of its key metrics and insights.

**Steps to Generate a Dashboard Summary**:
1. Select **Dashboard** from the dropdown menu in the Unified AI Agent interface and ask a query to generate a summary
2. Type the `@` symbol to view the list of available dashboards.
![Unified AI](/static/assets/artificial-intelligence-and-machine-learning/images/unified-ai/dashboard/adddashboard.png)
3. Select a required dashboard from the list to continue the query.
![Unified AI](/static/assets/artificial-intelligence-and-machine-learning/images/unified-ai/dashboard/addeddashboard.png)
4. Send the query to generate a summary.
![Unified AI](/static/assets/artificial-intelligence-and-machine-learning/images/unified-ai/dashboard/summary.png)

>**Note**: Users can submit queries only for dashboards to which they have access permissions.
