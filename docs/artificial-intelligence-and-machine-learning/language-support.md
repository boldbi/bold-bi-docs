# Language Support

Bold BI AI features support multiple languages, allowing you to interact with AI Assistant, AI Agent, AI Copilot, Dashboard Summary, and Widget Summary in your preferred language. To enable multilingual capabilities, you must configure the language settings in your [Bold BI application](/localization/latest/).

## Getting Started with language support

1. After configuring the language, open or refresh the AI Assistant page. You will see both the UI text and the AI responses presented in the selected language. 
   ![AI-Assistant-Sample1](/static/assets/artificial-intelligence-and-machine-learning/images/language-support/ai-assistant-sample1.png)

   ![AI-Assistant-Sample2](/static/assets/artificial-intelligence-and-machine-learning/images/language-support/ai-assistant-sample2.png)

   ![AI-Assistant-Sample3](/static/assets/artificial-intelligence-and-machine-learning/images/language-support/ai-assistant-sample3.png)

2. After refreshing your dashboard the AI Agent, AI Copilot, Dashboard Summary, and Widget Summary will reflect the selected language.  
   ![AI-Agent-Sample1](/static/assets/artificial-intelligence-and-machine-learning/images/language-support/ai-agent-sample1.png)

   ![AI-Copilot-Sample](/static/assets/artificial-intelligence-and-machine-learning/images/language-support/ai-copilot-sample.png)


## Troubleshooting

- If the AI features do not reflect the selected language, clear the browser cache and reload the application, or use an incognito/private browsing window.

>**NOTE**: In Site Settings, Application Language controls the application's locale UI text. To ensure AI responses in the selected language, set the Data Language option as well.

   ![Site-Language-Configuration](/static/assets/artificial-intelligence-and-machine-learning/images/language-support/site-language-configuration.png)
