# Manage Permissions

Manage Permissions allows administrators to control what users and groups can access and perform within Bold BI. Permissions can be assigned at both the **resource level** (such as dashboards, data sources, and categories) and the **feature level** (specific actions within dashboards like exporting, viewing underlying data, and using parameters).

You can configure both types of permissions from the **Manage Permissions** page in the Users or Groups section.

## Resource Permission

Resource Permission lets you define access modes — such as Read, Write, and Delete — for different resources. These permissions determine who can view, edit, delete, or manage dashboards, data sources, and categories.

## Resource Feature Permission

Resource Feature Permission allows you to assign permissions for specific dashboard actions. These include exporting dashboards, viewing underlying data, using dashboard parameters, and more. Feature permissions give fine‑grained control over the actions available to each user or group.
