# Overview

The Bold BI Dashboard Mobile app allows you to access your Bold BI dashboards on the go. Navigate through your dashboards to get insights into your data and stay up to date with the latest data. Check out the list of key features in the Bold BI Dashboard Mobile app below.

## Key features

**Dashboard management:** Dashboards are organized under categories such as Favorites, Recent, Public, and All Dashboards.

**View dashboards:** The built-in Dashboard Viewer control allows users to view dashboards.

**Export:** Dashboards can be exported to image, PDF, Excel, and CSV formats.

## Supported operating systems

* iOS 9.0 or later, compatible with iPhone, iPad, and iPod touch.
* Android: Android versions 4.1 or later.
